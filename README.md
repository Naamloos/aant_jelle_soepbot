# aant_jelle_soepbot
Beep boop uw motie is verworpen

## Wat is dit?
Aant Jelle Soepbot is een bot voor onze prive Discord server waarmee we "moties" kunnen indienen. De bot maakt automatisch een poll aan, en post of deze aangenomen is of verworpen wanneer de poll eindigt.
